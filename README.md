# site-ssl-archive

本仓库用于归档和发布多个独立的 HTML 页面。所归档的页面内容主要涉及 SSL/TLS 相关的信息呈现、状态展示或说明文档，均以静态 HTML 文件形式保存。

## 项目简介

该仓库旨在集中存放一批与 SSL 相关的独立页面，便于长期保存与公开访问。所有页面均为纯前端静态文件，不依赖后端服务，可直接通过浏览器打开或部署至任意静态托管平台。

## 目录说明

```
.
├── index.html          # 归档列表 / 导航页
├── pages/              # 存放各独立 HTML 页面
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
└── assets/             # 公共资源（CSS、JS、图片等）
    ├── style.css
    └── script.js
```

- `index.html`：提供归档页面的索引或导航，方便浏览。
- `pages/`：每个文件代表一个独立归档页面，文件名通常反映其内容或用途。
- `assets/`：存放页面共用的样式、脚本或图片资源。

## 页面归档说明

- 每个页面均为独立完整的 HTML 文件，包含自身所需的样式与脚本（或引用公共资源）。
- 页面内容不涉及具体域名、网站或商业服务，仅作为 SSL 相关信息的静态存档。
- 归档页面可能包含但不限于：SSL 状态检查结果、证书信息展示、配置示例、说明文档等。
- 所有页面均为历史存档，不保证实时更新或与现行标准完全一致。
